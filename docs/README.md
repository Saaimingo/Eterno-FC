# Documentação do Eterno FC

Esta pasta reúne as decisões que guiam o jogo. Ela é parte do produto: qualquer mudança estrutural deve atualizar a documentação correspondente.

## Ordem de leitura

1. `eterno-fc-design-v1.md` — visão, escopo e decisões de produto.
2. `arquitetura-tecnica-proposta-v1.md` — divisão técnica proposta para a evolução do protótipo.
3. Matrizes de regulamentos — referência para cada calendário e classificação do mundo simulado.
4. Relatórios `match-engine-mp*.md` — decisões, calibrações e limites de cada marco do motor.
5. `match-engine-mp8-live-coaching.md` — contrato das decisões tomadas durante a partida.

## Regra de manutenção

Regulamentos do futebol mudam. Cada alteração deve entrar como um novo pacote de regras por temporada, preservando o histórico de carreiras anteriores.
